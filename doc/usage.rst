.. _usage:

Usage
=====

::

  sudoku-solver {OPTIONS}

    Solves the classical Sudoku game.

  OPTIONS:

      -h, --help                        Display this help
      -f[file], --file=[file]           The file containing the Sudoku to solve
      -a[algorithm], --algo=[algorithm] The algorithm that should be used to
                                        solve the Sudoku
